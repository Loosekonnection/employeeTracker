## Description 

*A summary of what this project is, and why it has been created:*

This project allows a company to view the following data within their MySQL database: <br> 
*   Employees (All catagories)
*   Employees by Department
*   Employees by Managers
*   Roles
*   Departments 
*   Utlised Budget by Department

It provide the ability for a company to manage the following within their MySQL database: <br>
*   Add new Departments (name must be unique, validation check will advise if not)
*   Add new Roles (name must be unique, validation check will advise if not)
*   Add new Employees
*   Edit an Employee's Role
*   Edit an Employee's Manager
*   Delete an Employee
*   Delete an Employee Role
*   Delete a Department

## Installation

*How to install this project:*

You can either clone this repository to a local directory or you can fork this repository from within GitHub. <br> Once cloned you will need to run ```npm install``` within the cloned directory to install the following dependencies: <br>
*  [MySQL](https://www.mysql.com)
*  [Inquirer](https://www.npmjs.com/package//inquirer)
*  [Console Table Printer](https://www.npmjs.com/package/console-table-printer)

<br> The [Usage](#usage) section also has a video that covers how to install the project as well as how to use it.
